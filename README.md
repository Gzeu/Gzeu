## 👤 About Me

> *Architect of autonomous systems at the intersection of Blockchain, AI Agents, and GameFi.*

I'm an **Independent Full-Stack & Blockchain Developer** based in **Bucharest, Romania** — building production-grade decentralized systems, agentic economies, and GameFi platforms.

- 🔗 **Blockchain / Web3** — Rust smart contracts, DeFi protocols, agentic on-chain commerce on **MultiversX (Supernova)**
- 🤖 **AI Agents** — Multi-agent orchestration, LLM-powered workflows, AI task escrow, on-chain agent economies
- 🎮 **GameFi** — Play-to-earn platforms, NFT game mechanics, procedural world generation
- 🌐 **Full-Stack dApps** — Next.js 14, TypeScript, NestJS, xPortal, GCP & Vercel
- 🔐 **Security** — AES-256-GCM cryptographic tooling, audit logs, secure DevOps
